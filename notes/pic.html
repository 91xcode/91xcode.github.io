可选参数

1）type = { 302,img,json,length }，默认302。

# 以img标签显示图片：https://img.btu.pp.ua/api/?type=img

# 以json显示图片信息：https://img.btu.pp.ua/api/?type=json

# 显示图片数量信息：https://img.btu.pp.ua/api/?type=length

2）id = { Num } 。(指定mode，mode=8随机返回Coser不适用)

访问https://img.btu.pp.ua/api/?id=数字，跳转至指定图片

如：https://img.btu.pp.ua/api/?type=img&mode=3&id=8996

如：https://img.btu.pp.ua/api/?type=json&mode=5&id=546

如：https://img.btu.pp.ua/api/?mode=2&id=250
